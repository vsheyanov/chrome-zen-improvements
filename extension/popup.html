📊 Tab Event Logger

Status: Active

This extension detects tabs opened via "New Tab" button and moves them to the front with auto-scroll.

✨ Features:

  • 🔍 Smart detection - Only moves tabs opened via New Tab button (not links/duplicates)
  • 📌 Auto-reorder & focus - Detected tabs move to front and get activated
  • 🚀 Auto-scroll - Tab bar scrolls to show new tabs, respects pinned tabs
  • 📋 URL copying with notification - Copy current tab URL with visual toast notification
  • 📊 Complete tab event logging with detection details
  • 🎯 Perfect for managing crowded browser windows

Open Developer Tools (F12) → Console to see detailed logs including:

  • New Tab button detection - See criteria analysis for each new tab
  • Tab movement - Only New Tab button clicks get moved to front
  • Tab switching, closing, and URL navigation events
  • Scroll behavior when managing many tabs

Test it: Use Ctrl+T vs clicking links to see the difference!

⌨️ Keyboard Shortcuts:

  • Cmd+Shift+C (Mac) / Ctrl+Shift+C - Copy current tab URL with toast notification

⚠️ Override Note: Cmd+Shift+C conflicts with Chrome's "Inspect Element". You may need to manually set this in Chrome → Extensions → Keyboard shortcuts if it doesn't work automatically.

🔄 Alternative Shortcuts if Override Doesn't Work:

  • Cmd+Shift+Y - Usually available
  • Cmd+Shift+K - Often free
  • Cmd+Option+C - Different modifier combination
